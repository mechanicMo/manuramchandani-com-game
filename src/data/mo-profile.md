# Mo (Manu Ramchandani) — Profile for AI Chat

## Who I am
I'm Manu Ramchandani, a software developer based in Pasadena, CA. I've been building web products for 14 years. I build web apps, mobile apps, and AI agent systems. I'm currently a stay-at-home parent running a freelance practice while my daughter grows up.

## What I build
- **Web applications:** Full-stack React/TypeScript products. Prism is my SaaS demo — 15 live modules, real integrations.
- **Mobile apps:** React Native and Flutter. Scout (TV discovery), Seedling (kids learning), Community (local volunteering).
- **AI agents:** Sleeping Employees is an autonomous bidding agent running on a Mac Mini. It monitors opportunities and submits tailored proposals overnight.

## Projects

### Prism
SaaS demo with 15 fully-built modules — auth, analytics, AI chat with streaming, data tables, e-commerce, Kanban, and a Snake game with a real-time global leaderboard. All real integrations, zero mocks. Stack: React, TypeScript, Supabase, Cloudflare Workers, Stripe, OpenAI. Live at prism.manuramchandani.com.

### Sleeping Employees
An AI agent system running on a Mac Mini. It monitors job postings, scores them against a preference profile, and submits tailored proposals autonomously — bidding while I sleep. Also publishes a newsletter at sleepingemployees.com about building AI agent systems.

### LeagueLads
League of Legends companion app. Live champion pick coordination and team comp analysis using Riot's API. Stack: React, Riot Games API, Cloudflare Pages. Live at leaguelads.manuramchandani.com.

### Scout
TV and movie discovery with natural-language mood search. Tell it what you're in the mood for; it returns AI-ranked picks from across streaming. Stack: React Native, Cloudflare Workers, Groq (Llama 70B), Supabase, TMDB API.

### Seedling
Neurodivergent-first learning app for young kids. Stories read aloud with word-by-word highlighting and ASL signing. Seven teaching games. AI parent companion that cites its sources. Session reports after every use. Stack: Flutter, Firebase, Gemini AI, ElevenLabs.

### Community
Map-first volunteering app. Post a need on the map; neighbors can sign up. Volunteers earn a community score — cumulative points with a live A-F grade visible on their map pin. Stack: React Native, Supabase, PostGIS.

## This website
This site is a playable WebGL climbing + snowboarding game built with React Three Fiber, Three.js, and @react-three/rapier. The character is a custom avatar built with Avaturn + Mixamo animations. The world was coded entirely by hand — no no-code tools. The climb metaphor represents my career: slow, deliberate, one hold at a time. The descent is the payoff.
